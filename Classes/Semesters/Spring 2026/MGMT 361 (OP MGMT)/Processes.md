Businesses can output goods or services.

If process is prviding good - turns raw mats into product
if providing service - turns customer in need into a served customer

input - typically part of component that is incorporated in output.
	RAW MATERIALS

resource is used to create output but is not actually a part of the output.
	Things used to CREATE final output

![[Pasted image 20260114093428.png]]

Goods
	example - mcdonalds
		output is burgers
		the raw materials they need is the input
			bun, patty, sauce, vegetables
		they also need resources
			employees, cooking equipment
	example - toyota
		input - raw materials - metal, rubber, plastic
		resources - how to take raw mats into car
			machines, employees, facility
Services
	example Disney World
		entertainment service
		not a tangible good
		input and output are customers
			input - customer who did not get service
			output - customer who got service
			resource - rides, facility, employees
	example - Mayo Clinic
		input - sick people
		output- healthy people
		resource - doctors, medical equipment, hospital, nurses

### Resource Capacity
Capacity of one resource: Maximum number of units the resource can produce in a given time

Capacity of resource = 1/ Activity time

Activity time = time to process one unit by one resource
	takes 1 sec to draw head by one person - activity time
	capacity would be 60 heads per minute
